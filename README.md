## 팀 소개

+ 팀 이름 : 세븐 11

+ 팀 소개 : 다양한 상황에서 유연하게 대처할 수 있는 열정적인 팀원들로 구성되어 있습니다. 세븐일레븐이라는 이름처럼, 우리는 항상 열려 있는 마음으로 도전에 임하고 있습니다. 함께 노력하고 성장하는 것을 추구하며, 이를 통해 최고의 결과물을 만들어내고자 합니다.
